Делегирование событий

В JavaScript делегирование событий - это когда мы используем один обработчик событий для группы элементов.

classList

classList позволяет добавлять, удалять или проверять наличие классов у элемента.

add()

Метод add добавляет класс: element.classList.add('new-class')

remove()

Метод remove удаляет класс: element.classList.remove('old-class')

contains()

Метод contains проверяет наличие класса и возвращает true/false.
